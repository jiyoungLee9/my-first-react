# Colors
디자인 시스템의 핵심 색상 팔레트입니다. 
모든 색상은 `_design_tokens.scss`에 정의된 변수를 참조하여 관리됩니다.

## 3. Usage Guide (SCSS)
퍼블리싱 작업 시에는 아래와 같이 **Semantic 변수** 사용을 권장합니다.

```scss
// Good: 의미 있는 변수 사용 (유지보수 유리)
.error-message {
  color: $text-error;
}

// Bad: 절대적인 색상 값이나 기초 변수 직접 사용
.error-message {
  color: #e02535; // 또는 $ref-red-700
}
````
